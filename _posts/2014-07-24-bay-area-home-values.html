These choropleth maps (made with D3!) consist of annual averages for home values by zip code, for the Bay Area's 281 zip codes in its nine counties (San Francisco, San Mateo, Santa Clara, Marin, Napa, Sonoma, Solano, Contra Costa, Alameda), for the specified years (five-year intervals with 2013 as the last year). (Mouseover areas to see zip codes, home values, and county). The color scheme is calculated based on the min and max values of the 2013, to provide a better comparison, rather than the min and max values for each year. I downloaded the zip code data as a shapefile from the Metropolitan Transportation Commission, and converted to a geojson, using GDAL and running ogr2ogr (apologies if I'm messing up the terminology here - instructions found here). The data is from Zillow, using the Zillow Home Value Index (ZHVI) for All Homes. Zillow's Home Value Index has the most granularity, at the zip code level, and does not include values of distressed homes: "Distressed sales are often reduced in price, and when they make up a larger portion of the market – as they have in recent years – can skew overall values wildly."

These maps show that zip codes in the area referred to as the Silicon Valley has the highest home values, as does San Francisco. The map darkens in redness from 1998 to 2013. The darkest red, or highest home value in 2013, is zip code 94027, in Atherton, CA in San Mateo County, with a 2013 annual average home value of $4.2 million. The lowest 2013 annual average is $160,575 K in Richmond, CA in Contra Costa County.
